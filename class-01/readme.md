Python Data Types
Python has several built-in data types categorized as follows:

1. Numeric Types
int → Integer (e.g., 10, -5, 1000)
float → Floating-point number (e.g., 3.14, -2.5, 0.0001)
complex → Complex number (e.g., 3 + 4j, 1 - 2j)
2. Sequence Types
str → String (e.g., "Hello", 'Python')
list → List (e.g., [1, 2, 3], ['a', 'b', 'c'])
tuple → Tuple (e.g., (1, 2, 3), ('x', 'y', 'z'))
range → Range (e.g., range(5), range(1, 10, 2))
3. Set Types
set → Unordered, unique collection (e.g., {1, 2, 3})
frozenset → Immutable set (e.g., frozenset({1, 2, 3}))
4. Mapping Type
dict → Dictionary (e.g., {'name': 'Alice', 'age': 25})
5. Boolean Type
bool → Boolean (e.g., True, False)
6. Binary Types
bytes → Immutable sequence of bytes (e.g., b'hello')
bytearray → Mutable sequence of bytes (e.g., bytearray(5))
memoryview → Memory view object (e.g., memoryview(b'abc'))
7. None Type
NoneType → Represents the absence of a value (e.g., None)
Python Special Keywords
Python has reserved words (keywords) that cannot be used as variable names:

1. Control Flow Keywords
if, elif, else → Conditional statements
for, while → Loops
break, continue, pass → Loop control
return, yield → Function return values
raise → Raise an exception
2. Exception Handling Keywords
try, except, finally → Handling exceptions
assert → Debugging tool for testing conditions
3. Function & Class Keywords
def → Define a function
lambda → Define an anonymous function
class → Define a class
4. Variable Handling Keywords
global → Declare a global variable
nonlocal → Access variables in an enclosing scope
del → Delete an object
5. Boolean & None Keywords
True, False → Boolean values
None → Represents a null value
6. Import & Module Keywords
import, from, as → Importing modules
with → Context manager for handling resources
7. Object-Oriented Programming Keywords
is → Identity comparison
in → Membership test (e.g., x in list)
not, and, or → Logical operators
8. Asynchronous Programming Keywords
async, await → Define asynchronous functions
9. Special Keywords
match, case → Pattern matching (Python 3.10+)
